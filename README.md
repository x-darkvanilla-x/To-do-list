# To-Do List Application

This is a simple To-Do List web application built using HTML, CSS, and JavaScript.

![Screenshot 2024-03-29 225631](https://github.com/x-darkvanilla-x/To-do-list/assets/122479676/e3d95fbb-6b03-427e-938c-b6bc1fa031d6)

## Overview

This application allows users to create, manage, and organize their tasks in a simple and intuitive way. Users can add new tasks, and delete tasks as needed.

## Features

- **Add Tasks**: Users can add new tasks to the list by entering task descriptions and pressing the "Add" button.
- **Edit Tasks**: Tasks can be edited whenever needed.
- **Delete Tasks**: Completed or unwanted tasks can be deleted from the list by clicking on the delete button associated with each task.
- **Responsive Design**: The application is designed to be responsive and work well on various screen sizes, including mobile devices.

## Technologies Used

- **HTML**: Used for structuring the content and layout of the application.
- **CSS**: Used for styling the appearance of the application.
- **JavaScript**: Used for adding interactivity and functionality to the application.

## Getting Started

To run the application locally:

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.
